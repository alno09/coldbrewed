# ☕ ColdBrewed

<p align="center">
  <img src="https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/coffee.svg" width="100" />
</p>

Minimal mobile app to log cold brew experiments.

---

## Features

* Create cold brew batch logs
* Store brewing parameters
* View experiment history

---

## Tech Stack

* React Native (Expo)
* SQLite (local database)

---

## Setup

```bash
npm install
npx expo start
```

---

## Scope

ColdBrewed is intentionally minimal.

No cloud.
No social features.
No recommendations.

Just logging.
